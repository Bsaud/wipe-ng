# Wipe-ng
A **DoD 5220.22-M Standard Compliant** Anti-Forensic Storage Sanitization Tool

## How to use:

    sudo apt install pv
    ## changing file permission
    chmod +x wipe-ng.sh
    ## running the tool script
    sudo bash wipe-ng.sh <Device to Sanitize>
